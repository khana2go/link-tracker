# link-tracker
Track Links

## Getting started locally

    - start mongo locally

        $ mongod --dbpath <db-dir-path>

    - clone fork the repository

    - Run npm install

        $ npm install

    - start link tracker - you will need admin access to start link tracker on 80

        $ node index.js


## Sample Link

    - http://localhost/?c=xyz&t=http://www.yahoo.com
